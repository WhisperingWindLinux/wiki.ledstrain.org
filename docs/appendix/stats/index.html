            Additional Stats #

            Additional stats can be included from these two sources:

            • Matomo
            • Google Search

            Post the link formatted per the API (here’s a demo one ).

            • The data must be accessible through a wget or curl command.
            • If a api-key is used, anonymous or a similar value can be used indicate it should be replaced by the api-key (eg $apikey)
            • The result should be a image, json or csv file. Ideally a PNG image format or a JSON file.
            • Post the link and what it does on the forum or as a wiki issue .
            User privacy is important! API requests that provides too much information about specific visitors will not be added.
            For example, visitor logs.

            User Privacy #

            LEDStrain respects user privacy and data is anonymized on collection. Data is collected to help pursue the mission of LEDStrain, as well as identifying its impact.

            What is collected? #

            Information revealed to every website is collected.
            Additionally, see here for additional information matomo collects.
            Personally identifying information such as a ip address or username is either anonymized or not collected.
